# 移动端电商商城 APP 高保真原型设计

请设计一个**移动端电商商城 APP 的高保真原型（High-fidelity Prototype）**，整体风格现代、简洁，符合主流电商产品（类似淘宝 / 京东）的移动端交互规范。仅需支持**移动端尺寸（iPhone 14 / 390×844 或类似尺寸）**。

## 一、整体设计要求

* **平台**：移动端 APP
* **设计类型**：高保真 UI 原型
* **布局方式**：移动端响应式布局（仅适配手机）
* **底部导航（Tab Bar）**：固定 4 个 Tab
* **语言**： 所有语言统一使用简体中文

  * 首页
  * 分类
  * 购物车
  * 我的
* **交互要求**

  * TabBar 可切换页面
  * 支持弹窗（规格选择 / 加入购物车）
  * 支持商品数量调整
  * 支持订单状态切换

---

# 二、主要页面设计

## 1. 首页（Home）

页面结构：

1. **顶部导航栏**

   * 搜索框（占据主要区域）
   * 可包含搜索图标和占位文字

2. **轮播 Banner**

   * 横向轮播图
   * 展示促销活动或广告

3. **功能快捷入口（可选）**

   * 图标式入口，例如：

     * 秒杀
     * 优惠券
     * 新品
     * 热卖

4. **热门商品列表**

   * 网格布局（2 列）
   * 商品卡片包含：

     * 商品图片
     * 商品标题
     * 商品价格
     * 销量或标签
     * 加入购物车按钮（可选）

---

## 2. 分类页（Category）

页面结构：

**左右分栏布局**

左侧：

* 商品分类导航
* 列表形式
* 当前选中分类高亮

右侧：

* 对应分类商品列表
* 商品卡片展示：

  * 商品图片
  * 商品名称
  * 商品价格

交互：

* 点击左侧分类
* 右侧商品列表切换

---

## 3. 购物车页（Cart）

展示用户加入购物车的商品。

功能包括：

商品列表：

* 商品图片
* 商品名称
* 商品规格
* 商品价格
* 数量调整（+ / -）

操作：

* 单选商品
* 全选商品
* 删除商品
* 修改数量

底部结算栏：

* 显示总价
* 结算按钮

点击**结算**进入订单确认页。

---

## 4. 我的页面（Profile / Account）

展示用户相关信息和功能入口。

页面模块：

**用户信息区域**

* 用户头像
* 用户昵称
* 登录 / 注册入口

**订单入口**

展示订单状态入口：

* 待发货
* 待收货
* 已完成
* 退款 / 售后

**功能列表**

* 我的订单
* 收货地址
* 客服支持（可选）
* 设置

**底部**

* 退出登录按钮

---

# 三、核心业务页面

## 5. 商品详情页（Product Detail）

页面结构：

1. **商品图片轮播**
2. **商品基础信息**

   * 商品名称
   * 商品价格
   * 商品销量
3. **规格选择入口**

   * 点击弹出规格选择弹窗
4. **商品评价列表**
5. **商品详情图文介绍**

底部固定操作栏：

* 加入购物车按钮
* 立即购买按钮

弹窗设计：

**规格选择弹窗**

包含：

* 商品规格（颜色 / 尺寸）
* 商品数量选择
* 确认按钮

---

## 6. 订单确认页（Checkout）

页面内容：

**收货地址**

* 收件人
* 手机号
* 地址

**商品清单**

* 商品图片
* 商品名称
* 商品规格
* 数量
* 价格

**价格明细**

* 商品金额
* 运费
* 优惠（可选）

**支付方式选择**

* 微信支付
* 支付宝
* 余额支付（可选）

底部：

* 总价
* 提交订单按钮

---

## 7. 我的订单页（My Orders）

展示用户订单列表。

顶部：

订单状态 Tab：

* 全部
* 待发货
* 待收货
* 已完成
* 退款 / 售后

订单卡片内容：

* 商品图片
* 商品名称
* 商品价格
* 数量
* 订单状态

操作按钮（根据状态变化）：

* 取消订单
* 确认收货
* 查看物流
* 申请退款

---

# 四、设计规范建议

设计风格：

* 现代电商风格
* 简洁、清晰
* 高可用性

建议：

* 主色：红色或橙色电商风格
* 圆角卡片设计
* 统一阴影层级
* 清晰按钮层级
* 图标统一风格

组件建议：

* 商品卡片组件
* 订单卡片组件
* 规格弹窗组件
* 底部 TabBar 组件
